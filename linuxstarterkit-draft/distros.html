Picking Your First Linux Distro

Picking your first distribution of linux can be a daunting task. This page will help clear things up and provide a list of distros to choose from.

  • What is a linux distribution?
  • Does it matter which one I use?
  • List of Distributions

What is a Linux distribution?

To be clear, "distro" is just short for "distribution".

(Hoffman 2016) In short a distribution is just a variation of linux configured and implemented differently. (4)

(Hoffman 2016)There are many many linux distros out there. Thousands of linux distributions have been developed by corporations, governments, non-profits, and hobbyists to fill specific niches (4). To illustrate this here is linux.picture's "Periodic Table of Distros" (Figure 1). To make matters worse the periodic table is actually a simplified and slimmed down picture of Linux distros. For a full family tree of linux distributions see DistroWatch's family tree image. It's really mind blowing to look at.

(Hoffman 2016)A primary difference between linux distributions is often how the user interacts with the system (8). To demonstrate, (Figure 2) below we have Ubuntu 18.04 (left) and OpenSUSE Tumbleweed (right). You can use the slider at the bottom to compare how their interfaces differ.


Ubuntu and OpenSUSE are examples of two different distros developed by two different companies, Canonical Ltd. and SUSE respectively. They take very different approaches to the user interface.(Jasuja, Gauri, Techboy317 2019) While both interfaces include most of the same features functionally, OpenSUSE uses the KDE interface, which according to diffen.com, follows a similar format of the Windows UI while Ubuntu uses the GNOME interface which has it's own philosophy and design for graphical user interfaces. source

Does it matter which distro I use?

Based on my experience it doesn't matter what your first Linux distribution is as long as it's user-friendly. Below is a list of distros that are safe for new users and a list of distros that new users should avoid. This list is based on my personal experience with Linux and hearing the stories of new users. However do note that there are a lot of distributions with a niche use case that you might like to use which I do not list. If you are interested in finding something specific for your use case such as a scientific Linux distro take a look at distrochooser, a website with a quiz which finds a distro that matches your use case.

List of Distros

Note: This list is too vast to link to all the distributions. To find one just google search the name or alternatively head on over to Distrowatch, where you can search and browse distros.

    Recommended

    For the purpose of this website I will recommend Kubuntu 18.04 LTS and I will include step by step instructions to installing it on the install page

  • Kubuntu 18.04 LTS

    Decent Hardware (last 5 years)

    These are distributions you could run on most hardware. If you have an old or minimal PC please refer to the next list.

  • Ubuntu (all the Ubuntu flavors. Kubuntu, Lubuntu Xubuntu etc.) (I recommend using LTS for best software support),
  • Linux Mint (Cinnamon Edition Recommended)
  • Fedora Linux
  • Manjaro Linux
  • ZorinOS (highly recommend the paid edition if you have the money. They provide you with good support and the money goes towards improving the linux desktop experience source)
  • Solus
  • Deepin

    Old Hardware

    These distros are lightweight and minimal. Excellent for old hardware and netbooks

  • Lubuntu
  • Xubuntu
  • WattOS R10 (not the microwatt edition) [According to the wattOS about page it is a distro focused on using less power and battery saving thus only use if you intend to get better battery life source]
  • SolydX
  • Linux Lite
  • PeppermintOS
  • Puppy Linux (only for REALLY old hardware)

    Distros To Avoid for New Users

    These distros in my experience are generally unfriendly to new users despite being recommended by long time users quite often

  • Arch Linux (Has no graphical installer)
  • default debian (not particularly difficult to use however lacks basic configuration)
  • Gentoo (no graphical installer, all software compiled from source)
  • Anything that uses the i3 desktop (the i3 desktop is fully keyboard controlled and has a learning curve)
  • CentOS (While usable as a desktop OS, CentOS is primarily for servers and has strict security constraints which make it difficult to use without knowlege)
  • Alpine Linux (server OS)
  • Damn Small Linux (extremely minimalist distro meant for hobbyists)
  • TInyCore (Similar to Damn Small)
  • Kali Linux (this is a distribution for ethical hacking not a general purpose OS)
  • Slackware
  • FreeBSD (Not user friendly and also not Linux)
